# Documentation
This is the index of the documentation for yaponmdl project.

## Topics list
A list of topics related to Patapon graphics file formats:
 - [GXP/GXX/GXT file format](./gxp_file_format.md) - graphics file format unique to Patapon franchise that can contain textures, drawables, and even very basic skeletal animations.
 - [GMO file format](./gmo_file_format.md) - generic model scenegraph file format from the official PSPSDK, that the Patapon series uses for more advanced skeletal animations.
